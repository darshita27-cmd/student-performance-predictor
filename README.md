# Student Performance Predictor

## Overview
This repository contains a Streamlit app designed to predict student final grades based on attendance and test scores. The app uses a linear regression model trained on synthetic data and includes a 3D visualization to show the relationship between inputs and predicted grades.

## Usage
- Input the attendance percentage and test scores.
- Click "Predict grade" to see the estimated final grade and a 3D scatter plot.

## Live App
([https://student-predictor.streamlit.app](https://student-performance-predictor-m6kndxcqety8fedkbbs8wp.streamlit.app/))

## Requirements
The app requires the following Python packages, listed in `requirements.txt`:
- `streamlit`
- `pandas`
- `numpy`
- `scikit-learn`
- `plotly`

## Code
